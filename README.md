# TauTable
A Program that will generate a truth table given a string of statements or propositions. It makes creating truth tables easier and more efficient while having the advantage of it being lightweight and fully offline.
## key features
- #### [Graphical User Interface](#graphical-user-interface-gui)
- #### [Built in Interpreter](#built-in-interpreter-logical-interpreter)
